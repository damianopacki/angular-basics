1. One way data binding.

String interpolation

Current title - {{ title }}

Interpolacja służy do wyświetlania w szablonie naszego komponentu (pliku html) zmiennych stworzonych w klasie tego komponentu (pliku ts). Do wyświetlania danych nalezy użyć podwójnych nawiasów klamrowych.

2. Property binding.

Zadanie:

Za pomocą property binding stwórz diva, który będzie miał wymiary 100x100 i zielony background.

Zadanie:

Za pomocą interpolacji tekstu wypisz w widoku sowje imię i wiek. Imię i wiek umieść w polach klasy.

My name: {{ name }}

My age: {{ age }}

3. Event binding.

Łączenie zdarzeń to przypisywanie metod do określonych właściwości elementów html za pomocą nawiasów '()'.

Zadanie:

Stwórz w klasie pole, które będzie przechowywało liczbę. Stwórz button, który po kliknięciu będzie inkrementował stworzoną liczbę. Wyświetl przechowywaną liczbę w widoku.

Count: {{ countNumber }}
